# Hangman Game (Python)

A simple console-based Hangman Game built using Python.

## Features
- Uses a predefined list of 5 words.
- Randomly selects a word for each game.
- Allows a maximum of 6 incorrect guesses.
- Displays guessed letters and remaining attempts.
- Simple command-line interface.

## Concepts Used
- random module
- while loop
- if-else statements
- strings
- lists

## How to Run

1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open the terminal in the project folder.
4. Run the program:

python hangman.py

## Game Rules

- Guess the hidden word one letter at a time.
- If the guessed letter is correct, it will be revealed in the word.
- If the guessed letter is incorrect, one attempt is lost.
- You have only 6 incorrect guesses.
- Guess the complete word before running out of attempts to win.
